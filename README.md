# BountyScope

BountyScope is a security research assistant for ethical hackers and bug bounty hunters.

## Features

- HTTP Request Analyzer
- IDOR detection
- JWT detection
- Automatic parameter extraction
- Payload suggestion engine
- CVSS scoring (coming soon)
- Bug bounty report generator (coming soon)

## Tech Stack

Frontend
- React
- Vite
- TailwindCSS

Backend
- Node.js
- Express

## Usage

Start the server:

npm install
npm run dev

Open:

http://localhost:5173

Paste an HTTP request and analyze it.

## Ethical Use

BountyScope must only be used on systems you are authorized to test.
